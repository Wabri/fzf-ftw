# fzf-ftw

## How to use

Clone into .local/bin:

```
git clone https://github.com/Wabri/fzf-ftw.git ~/.local/bin/fzf-ftw/
```

Add to your shell configuration:

```
source ~/.local/bin/fzf-ftw/*.sh
```

Or source when you need it.
