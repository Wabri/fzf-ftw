# fzf-ftw

## How to use

Clone into .local/bin:

```
git clone https://github.com/Wabri/fzf-ftw.git ~/.local/bin/fzf-ftw/
```

Add to PATH on .bashrc or .zshrc:

```
export PATH="$HOME/.rvm/bin:$HOME/.local/bin/fzf-ftw/:$PATH"
```
